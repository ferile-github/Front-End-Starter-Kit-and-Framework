Responsive Grid

This is a responsive grid system, using a base of 12 columns. You can set columns to behave differntly across different breakpoints, by using the prefixed classnames.


.col-* 		Create a column for all breakpoints, excluding mobile devices.
.col-lg-* 	Create a column for Desktop browsers
.col-md-* 	Create a column for Tablet landscape and medium Desktops
.col-sm-* 	Create a column for Tablet portrait and small Desktops
.col-xs-* 	Create a column for Mobile devices

Offsets

Offset a column


<div class="row">
	<div class="col-offset-3 col-9">
		<div class="box">offset</div>
	</div>
</div>

Nested Grids

Nest grids inside grids inside grids.


<div class="row">
	<div class="col">
		<div class="box">auto</div>
			<div class="row">
				<div class="col">
					<div class="box">auto</div>
				</div>
			</div>
		</div>
	</div>
</div>

Alignment

Add classes to align elements to the start or end of a row as well as the top, bottom, or center of a column

.start


<div class="row start">
	<div class="col-6">
		<div class="box">
			start
		</div>
	</div>
</div>

.center


<div class="row center">
	<div class="col-6">
		<div class="box">
			start
		</div>
	</div>
</div>

.end

<div class="row end">
		<div class="col-6">
			<div class="box">

		</div>
	</div>
